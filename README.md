# TaskFlow — Premium Dark Theme Task Manager

An ultra-premium dark-themed task manager with particle animations, glassmorphism design, and smooth interactions.

## Features

- Add tasks (Enter key or button)
- Toggle completion with animated checkboxes
- Delete tasks with hover-reveal buttons
- Filter: All / Active / Completed
- Live task counter
- Canvas particle system with connections
- Glassmorphism card with backdrop blur
- Slide-in task animations

Open `index.html` in a browser.
